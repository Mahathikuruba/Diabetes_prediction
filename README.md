# 🩺 Diabetes Prediction using Machine Learning

## 📌 Project Overview

This project predicts whether a person is likely to have diabetes using Machine Learning techniques. The model is trained on a medical dataset containing features such as glucose level, blood pressure, BMI, insulin, and age.

The goal of the project is to build a simple and effective diabetes prediction system using Python and Scikit-learn.

---

## 🚀 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

---

## 📂 Dataset

The project uses the **Pima Indians Diabetes Dataset**.

### Features Used

* Pregnancies
* Glucose
* BloodPressure
* SkinThickness
* Insulin
* BMI
* DiabetesPedigreeFunction
* Age

### Target

* **Outcome**

  * `0` → Non-Diabetic
  * `1` → Diabetic

---

## ⚙️ Project Workflow

### 1. Data Loading

The dataset is loaded using Pandas.

### 2. Data Preprocessing

* Checked for missing values
* Cleaned the dataset
* Separated features and target variable

### 3. Train-Test Split

The dataset is divided into training and testing sets.

### 4. Model Training

A Machine Learning classification model is trained using the training data.

### 5. Model Evaluation

The model is evaluated using accuracy and prediction results.

### 6. Prediction

The trained model can predict whether a new patient is diabetic or not.

---

## 📊 Output Example

**Input:** Medical details of a patient

**Output:** `Diabetic` or `Non-Diabetic`
